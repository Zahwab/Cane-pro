# Security Policy

## Supported Versions

Currently, only the latest `main` branch is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously.

If you discover a security vulnerability in Cane-Pro, please do **not** open a public issue. Instead, please report it via private email to the active maintainers (or through GitHub's standard Security Advisory reporting features if enabled).

We will attempt to acknowledge your report within 48 hours and work with you to diagnose and patch the issue as quickly as possible.
